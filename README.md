# 🤖 botFrases - WhatsApp Motivacional

Bot que envia frases motivacionais automáticas para o WhatsApp usando a biblioteca WWBJS e a API ZenQuotes.

---

## 🔌 Tecnologias utilizadas

- [Node.js]
- [whatsapp-web.js]
- [ZenQuotesAPI]
- [Google_TranslateAPI(não_oficial)]
